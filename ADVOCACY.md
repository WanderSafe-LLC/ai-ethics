# AI Exploitation Advocacy Resources

This document lists organizations, websites, and social media accounts actively fighting against AI exploitation of creative works. The focus is on demanding attribution, payment, and protections for developers, artists, writers, and all workers impacted by AI.

Last updated: March 1, 2026

## Advocacy Websites & Organizations

### Major Organizations

- **Electronic Frontier Foundation (EFF)**: Advocates for digital rights, including AI copyright and fair use. Website: [eff.org](https://www.eff.org)
- **Creative Commons**: Promotes open licensing and fights against unauthorized AI training on creative works. Website: [creativecommons.org](https://creativecommons.org)
- **Authors Guild**: Represents writers and advocates for copyright protections against AI. Website: [authorsguild.org](https://www.authorsguild.org)
- **Screen Actors Guild - American Federation of Television and Radio Artists (SAG-AFTRA)**: Union fighting for performers' rights in AI-generated content. Website: [sagaftra.org](https://www.sagaftra.org)
- **Writers Guild of America (WGA)**: Advocates for writers' rights in the age of AI. Website: [wga.org](https://www.wga.org)
- **Artists Rights Society**: Protects visual artists' rights, including against AI reproduction. Website: [artistsrights.org](https://www.artistsrights.org)
- **Have Been Trained**: Coalition of artists suing AI companies for copyright infringement. Website: [havebeentrained.com](https://havebeentrained.com)
- **Glaze**: Tool and organization protecting artists from AI style mimicry. Website: [glaze.cs.uchicago.edu](https://glaze.cs.uchicago.edu)
- **Nightshade**: Poisoning tool for AI training data to protect artists. Website: [nightshade.cs.uchicago.edu](https://nightshade.cs.uchicago.edu)
- **Spawning**: AI detection and protection tools for artists. Website: [spawning.ai](https://spawning.ai)
- **Class Action Lawsuits**: Various lawsuits like Andersen v. Stability AI, Thaler v. Perlmutter, etc. Resources: [wikipedia.org/wiki/Copyright_aspects_of_artificial_intelligence](https://en.wikipedia.org/wiki/Copyright_aspects_of_artificial_intelligence)
- **OpenAI Copyright Lawsuit**: Led by authors and artists. Updates: [theverge.com](https://www.theverge.com)
- **Stability AI Lawsuits**: Multiple cases from artists. Updates: [artnews.com](https://www.artnews.com)
- **Midjourney Lawsuits**: Artist coalitions. Updates: [hyperallergic.com](https://hyperallergic.com)
- **Developer Advocacy Groups**: Groups like All Developers Alliance pushing for AI code attribution.

### Additional Resources

- **AI Ethics Guidelines**: Various frameworks from IEEE, ACM, etc.
- **Worker Unions**: AFL-CIO statements on AI impact on jobs.

## Template Posts for Activists

Use these templates to create social media posts demanding payment and attribution from AI companies.

### X (Twitter) Templates

1. **Demand Payment**: "AI companies are making billions from our creative work without paying us. Demand fair compensation now! #PayCreators #AIExploitation [Link to lawsuit]"

2. **Attribution Fight**: "Your art, code, writing – stolen by AI without credit. Fight back for attribution rights! Join the movement. #HumanCreativity #AITheft"

3. **Worker Solidarity**: "AI isn't just stealing from artists – it's threatening all jobs. Developers, writers, workers: unite against exploitation! #WorkersRights #AI"

4. **Call to Action**: "Contact your representatives: AI companies must pay for training data. Your work deserves payment! #AIReform #CreatorRights"

5. **Share Story**: "Share how AI exploited your work. Together, we demand payment and attribution. #MyStory #AIJustice"

### LinkedIn Templates

1. **Professional Advocacy**: "As a developer/artist/writer, I've seen AI exploit our work without compensation. It's time for industry standards on attribution and payment. #AIethics #IntellectualProperty"

2. **Network Call**: "Connect with me if you're fighting AI exploitation. Let's build a coalition for fair treatment of creative professionals. #Networking #AIAdvocacy"

## GitHub Action for Updates

To keep this list dynamic, add the following GitHub Action workflow to update follower counts and add new advocates regularly.

Create `.github/workflows/update-advocacy.yml`:

```yaml
name: Update Advocacy List

on:
  schedule:
    - cron: '0 0 * * 0'  # Weekly on Sunday
  workflow_dispatch:

jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Update follower counts
        run: |
          # Script to fetch current follower counts from X API or web scraping
          # This requires API keys or scraping tools
          python scripts/update_followers.py
      - name: Commit changes
        run: |
          git config --local user.email "action@github.com"
          git config --local user.name "GitHub Action"
          git add ADVOCACY.md
          git commit -m "Update advocacy list follower counts" || echo "No changes"
          git push
```

*Note: Implement `scripts/update_followers.py` using Twitter API v2 or web scraping libraries like BeautifulSoup. Requires API keys for accuracy.*
